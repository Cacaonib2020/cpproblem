# F - What's GCD

## 問題文

標準入力から自然数A,Bが入力されます。
これらの最大公約数を出力してください。

## 制約
* ***A***,***B***は自然数
* 1≦***A***,***B***≦10000
***
## 入力
入力は以下の形式で標準入力から与えられる．

<pre>
<span style="font-size: 200%"><var>A</var> <var>B</var>
</pre>
## 出力
AとBの最大公約数を１行で出力してください。。
***
## 入力例 1 
```
10 6
```
## 出力例 1
```
2
```

***
## 入力例 2
```
250 250
```
## 出力例 2
```
250
```
***
## 入力例 3
```
8 79
```
## 出力例 3
```
1
```
***
